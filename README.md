# README

* This project was made using Ruby on Rails, React.js, PostgreSQL and Foundation

* Ruby version 2.3.3

* Gems:
  Devise,
  ActionMailer

* Database creation:
  `rake db:create`

* Database initialization:
  `rake db:migrate`

* Run the test suite:
  `rake`
